<h1>IDX How-To &amp; Help</h1>
<h2>Bookmarked Listings</h2>
<h3>Overview</h3>
<p>Bookmarked listings are listings that you want to save and view later.</p>
<h3>Bookmark a Listing</h3>
<p>To Bookmark a Listing click the star icon near each listing result or the Bookmark Listing button found on each listing's details page.</p>
<hr />
<h2>Saved Searches</h2>
<p>If you find yourself searching for the same types of properties all the time, you needn't fill out the search form each time--create a saved search for one click access to search results.</p>
<h3>Save a Search</h3>
<p>To Save a Search Click the "<strong>Save this Search!</strong>" link found at the top of any listings results page.</p>
<h3>Daily Listing Updates</h3>
<p>New listings matching your saved search criteria will be sent to you each day. This will keep you informed on the latest properties available.</p>
